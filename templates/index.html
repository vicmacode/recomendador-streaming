🔗 Agrupación de usuarios

Encuentra grupos de usuarios con gustos muy parecidos. Si dos personas califican las mismas películas de forma similar, quedan en el mismo grupo.

UFDS (Quick-Union Ponderado). Umbral: similitud coseno ≥ 0.99, mínimo 20 películas en común. Complejidad: O(α(n)) por operación.

Haz clic para analizar

🌐 Comunidades de influencia

Detecta comunidades donde los usuarios se influyen mutuamente: si a ti te gusta lo que le gusta a otro, y viceversa, forman una comunidad.

SCC — Algoritmo de Kosaraju. Grafo dirigido de influencia + DFS en grafo transpuesto. Complejidad: O(V + E).

Haz clic para analizar

🌳 Red mínima de conexiones

Conecta a todos los usuarios similares usando la menor cantidad de enlaces posible, como un mapa de relaciones simplificado.

MST — Algoritmo de Kruskal. Peso = (1 - similitud coseno). Usa UFDS para detección de ciclos. Complejidad: O(E log E).

Haz clic para analizar

🔀 ¿Cómo repartir películas entre usuarios?

Imagina que solo puedes recomendar un número limitado de películas. ¿Cómo las repartes entre varios usuarios para que todos reciban algo que les guste? Este módulo encuentra la distribución óptima.

Flujo Máximo — Ford-Fulkerson (Edmonds-Karp). Red: Fuente → Usuarios → Películas → Sumidero. BFS para caminos aumentantes. Complejidad: O(V × E²).
